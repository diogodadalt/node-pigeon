## node-pigeon

[![Build Status](https://travis-ci.org/hollywoo/node-pigeon.svg?branch=master)](https://travis-ci.org/hollywoo/node-pigeon)

A node package to implement microservices reliability patterns including
- Timeouts
- Retries
- Fallbacks
- Circuit Breaker
- Jittering
- Agent pooling

#### Reference
- Netflix
- Google SRE
- Distributed Systems
